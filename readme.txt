== Description ==


The Simple Pinterest Feeds is an awesome tool for your Wordpress website, enabling you to post a feed directly from Pinterest on your pages! The Pinterest feeds is easy to install, easy to use and provides users with an ample amount of benefits.

Take a look at the features and benefits sure to be appreciated with the Simple Pinterest Feeds.

i) Options – There are tons and tons of options available for your Pinterest pins with the Simple Pinterest Feeds. You can easily customize the Simple Pinterest Feeds to your specifications, which includes the width and the height of the slider on your page.


ii) Compatible –The Wordpress Pinterest Feeds is compatible with all browsers, including Chrome, Safari, Firefox and Internet Explorer

	 
iii) Use Shortcode

Easy to use On Different pages or posts or widget using shortcode "[sptf_feeds_output]".

Use This Code in Page Template <code> <?php echo do_shortcode('[sptf_feeds_output]')

== Installation ==



This section describes how to install the plugin and get it working.

1. Install the wordpress plugin first

You can install it via Wordpress Installer or can download the file first then Upload it on /wp-content/plugins/directory

2. Activate the plugin

3. You will see pinterest settings from right side of your wordpress menu.

Configuration Options:

Pinterest Username : Default is "pinterest" - You can change that with your pinterest username

Margin Top - Height & Width : Their you have to Put Numeric Number as per your need.

Image scale : Default is 120 - Change it with a numeric value you prefer.
